# Magyar Filmek - online film megtekintő webalkalmazás
Ez a projekt egy online film megtekintő webalkalmazás ami a magyar filmekre specializálódott. A felhsználók magyar filmeket tekinthetnek meg ezen a felületen, elmondhatják vélemnényüket és értékelhetik őket valamint lehetőséget biztosít az adminisztrátorok számára hogy kezelhessék és a bővíthessék a film készletet.

## Funkciók
### Vendég (guest) funkciók 
- Regisztráció és bejelentkezés
- Elérhető filmek megtekintése
- Vélemények megtekintése (szöveges)
- Értékelések megtekintése

### Felhasználó (user) funkciók
- Bejelentkezés
- Elérhető filmek megtekintése
- Vélemények megtekintése (szöveges)
- Értékelések megtekintése
- Vélemények írása
- Értékelések írása

### Admin (admin) funkciók
Bejelentkezés
- Elérhető filmek megtekintése
- Vélemények megtekintése (szöveges)
- Értékelések megtekintése
- Vélemények írása
- Értékelések írása
- Új filmek hozzáadása
- Felhasználók kezelése
- Vélemények moderálása (ha szükséges)
- Teljes rendszer áttekintése
